Setup Ethernaut

August 18, 2023 Â· 3 min Â· Ardias

Halo semuanya, jadi pada post pertama ini saya akan membagikan cara untuk setup Ethernaut secara lokal (Linux).

Jadi sebelum kita melakukan setup, saya akan membagikan beberapa versi tools dan software yang saya gunakan untuk bermain di Ethernaut yang dapat berjalan di lokal saya yaitu:

  1. Node.js v18.17.1
  2. NPM v9.8.1
  3. yarn
  4. git

Steps#

  1. Clone repo dan install dependencies
git clone https://github.com/OpenZeppelin/ethernaut.git && yarn install
  1. Setelah terinstall, jalankan ganache atau rpc servernya pada folder ethernaut
yarn network
  1. Lanjut kita akan melakukan installasi Metamask sesuai browser yang digunakan
  2. Setelah Metamask terinstall, selanjutnya kita akan menambahkan network baru dengan klik pada bagian pojok kiri atas lalu klik Add Network dan setelah di klik pilih Add Network Manually
  1. Isi kolom yang kosong menjadi seperti berikut dan untuk Network name bebas diisi sesuai kemauan kalian
  1. Setelah itu kita akan memasukkan akun yang dari Ganache, jadi kita memerlukan private key yang ada pada output Ganache di terminal
  1. Untuk Import Account yang ada di Ganache ini kita bisa klik Account 1 atau yang bagian atas yang paling tengah, lalu akan nanti akan muncul sebuah poppup dan pilih Import account
  1. Masukkan private key yang sudah di copy dari Ganache ke kolom berikut ini, lalu Import

Tampilan jika berhasil melakukan import account

  1. Setelah Metamask ready, kita akan pindah untuk melakukan deploy dan menjalankan servernya nanti. Tetapi sebelumnya kita pergi ke file client/src/constant.js untuk melakukan uncomment pada line 180 agar site nya bisa dijalankan menggunakan NETWORK.LOCAL
  1. Pindah lagi ke atas yaitu di line 16 yaitu url dari localhost menjadi http://127.0.0.1

  1. Langsung saja deploy contract dengan menjalankan perintah sebagai berikut
yarn deploy:contracts

Nanti akan muncul Confirm deployment dan bisa ketik y saja

Berikut tampilan jika deploy contract berhasil

  1. Sebelum menjalankan server/clientnya, kita perlu menambahkan dahulu pada file package.json di bagian script start dan buildnya (melanjutkan saja) karena terdapat error juga
export SET NODE_OPTIONS=--openssl-legacy-provider &&
  1. Baru kita bisa memulai server/client dengan menjalankna perintah berikut ini
yarn start:ethernaut
  1. Jika kita baru menjalankan pertama kali maka nanti akan muncul popup dari Metamask seperti berikut untuk memilih akun yang telah kita import dari ganache agar bisa terhubung ke site Ethernaut lokal kita.

Selanjutnya tinggal pilih Connect saja

  1. Nanti jika sudah selesai maka akan muncul sebuah popup seperti berikut, cukup klik Deploy game
  1. Dan terakhir akan muncul sebuah popup dari Metamask untuk konfirmasi transaksinya

Yep setelah itu kita tinggal memainkan level mana saja yang kita inginkan.

Sebenarnya setup seperti ini tidak efektif karena pasti bakal terjadi error. Jadi perlu diingat untuk memulai level ini kembali lakukan deploy kembali dan ganti account di Metamask menggunakan private key yang lain jangan sama seperti sebelumnya.

Mungkin segitu saja pada post pertama saya ini yaitu melakukan setup Ethernaut agar bisa dijalankan secara lokal.
